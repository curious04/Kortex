Welcome to Kortex

note 2026-07-17 getting-started

This is your knowledge hub — a calm place to keep notes, links, ideas, tasks, routines and files. Everything is a simple Markdown file, so your data is yours forever (no lock-in).

How it works

  • Every card on the home page is one file in content/notes/.
  • Add a small header (frontmatter), write anything below it, save — done.
  • Push to GitHub and it deploys itself to a live website you can open anywhere.

The magic

  • Only you can add directly (you push to main).
  • Anyone else can suggest additions via a Pull Request — you approve, it merges.
  • Everything is versioned — you can never truly lose a note.

Tip: pin the things you use most with pinned: true.
